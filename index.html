💡 Quick Tips

📱
Add to home screen — Open in mobile browser → Share → Add to Home Screen for a full app experience
🔄
Widget syncs live — The task widget shares data with this app. Complete a task in the widget and it updates here too
🎨
Widget themes — Go to Settings → Home Screen Widget to choose your widget card style
🔔
Reminders work best on HTTPS — Host on GitHub Pages (free) for notifications and the app icon to work properly
📅
Calendar entries — You can add multiple entries per day. Each saves separately with its own mood and note
